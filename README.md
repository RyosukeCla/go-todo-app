# Go todo app
simple todo app for learning go

## stack
- go 1.11.1
- react 16.5.2

## start server
### mac: prebuild binary
```bash
$ ./go-todo-app
```

### install requirements
```bash
$ sh ./scripts/install.sh
```

### dev
```bash
$ sh ./scripts/dev.sh
# go to localhost:3000
```

### build
```bash
$ sh ./scripts/build.sh
$ ./go-todo-app
```
